# GitHub Pages 上线（只需 GitHub Desktop）

**不需要 Netlify，也不需要 Netlify Desktop。**

## 第 1 步：在 GitHub Desktop 添加这个文件夹

1. 打开 **GitHub Desktop**
2. **File → Add local repository**
3. 选择文件夹：  
   `C:\Users\timot\Documents\I Have a Plan\ihcr-diary`
4. 若提示不是仓库，选 **create a repository** 并继续

## 第 2 步：发布到 GitHub

1. 点右上角 **Publish repository**
2. Name 建议：`ihcr-diary`（或你喜欢的英文名）
3. 勾选 **Keep this code public**（Public 才能免费用 Pages）
4. 点 **Publish repository**

## 第 3 步：开启网站（GitHub 网页）

1. 浏览器打开你刚建的仓库（GitHub Desktop 里点 **View on GitHub**）
2. **Settings → Pages**
3. **Build and deployment → Source** 选 **Deploy from a branch**
4. **Branch** 选 `main`，文件夹选 **`/ (root)`**，点 **Save**
5. 等 1–3 分钟，会出现绿色链接，例如：  
   `https://你的用户名.github.io/ihcr-diary/`
